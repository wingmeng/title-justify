font-style：设置字体样式

  • font-style:normal 规矩的字体
  • font-style:italic 倾斜的字体

font-variant：设置字母的大小写样式

  • font-variant:normal ABCabc （正常的字母）
  • font-variant:small-caps ABCabc （小写字母变大写，体积变小型）

font-weight：设置文本的粗细

  • font-weight:normal 标准粗细
  • font-weight:bold 加粗字体
  • font-weight:bolder 再次加粗字体
  • font-weight:400 使用数字值(100-900)（等价于normal）
  • font-weight:700 使用数字值(100-900)（等价于bold）

font-size：设置字体大小

  • font-size:16px 使用px作为单位（绝对大小）
  • font-size:1.2em 使用em作为单位（相对大小）
  • em相对于父容器字体大小而改变。默认值：1em=16px

lline-height：设置字体行高

  • line-height:110%
    基于当前字体尺寸的百分比行间距。
  • line-height:2.2
    设置数字，此数字会与当前的字体尺寸相乘来设置行间距。
  • line-height:20px
    使用像素，设置固定的行间距。

font-family：设置字体（通用字体）

  • font-family:Serif 字体成比例，而且有上下短线。
  • font-family:Cursive 模仿人的手写体。
  • font-family:Sans-serif 成比例，而且没有上下短线。
  • font-family:Monospace 字体不成比例。通常用于模拟打字机文本。
  • font-family:Cursive 个性化字体。
  • font-family:\5B8B\4F53 宋体
  • font-family:\6977\4F53 楷体
  • font-family:\5FAE\8F6F\96C5\9ED1 微软雅黑

组合技：

  • font:italic normal bold 14px/150% '\5FAE\8F6F\96C5\9ED1'
    倾斜非异体加粗14号150%行距微软雅黑字体。
